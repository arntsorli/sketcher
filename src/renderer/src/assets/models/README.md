# Bundled model manifest

These small GLB files replace the most visible procedural placeholders at render time. Sketcher keeps the procedural versions as fast placement previews and as a loading fallback. Models are converted from glTF's Y-up convention to Sketcher's Z-up scene and scaled to a useful real-world size in `SceneCanvas.tsx`; the source files themselves are unchanged.
